# Public Demo Template

Use this when preparing a demo for GitHub, YC applications, or public writeups.

## Demo Prompt

- one-sentence premise
- target benchmark card
- target reader effect

## Loop Trace

- baseline output
- judged failure
- selected intervention
- revised output
- score delta

## Demo Claim

State one narrow claim only.

Good:

- "the system improved chapter-end propulsion from 2 to 4"
- "the system repaired clue fairness without losing mystery"

Bad:

- "the system writes top novels now"
