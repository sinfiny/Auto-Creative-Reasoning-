# Source Notes

This skill is primarily grounded in the Daniel Pink interview and secondarily supported by online-writing workflow material cited in `David-Perell-writing-skills-proof.md`.

Core structural moves:

- identify the governing question before building sections
- define what each section must do for the reader
- arrange sections so later sections mean more because of earlier ones
- distinguish a real structure from a pile of adjacent topics

Use this when the outline is the problem. If the issue is sentence quality, route elsewhere.
