# Loop

This folder defines the operating contracts for autoresearch-style creative runs.

The point of the loop is not endless rewriting. The point is controlled iteration with explicit stop conditions.

## Loop Stages

1. choose target unit
2. choose benchmark card
3. choose rubric sections
4. draft or revise
5. judge
6. decide rewrite level
7. apply one intervention
8. compare deltas
9. stop or continue

Use the files in this folder to keep runs comparable.
