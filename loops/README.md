# Loops

This folder defines the operating system around autoresearch-style creative runs.

The point of the loop is not endless rewriting. The point is controlled iteration with explicit stop conditions, clear preset shapes, and explicit candidate additions.

## Folder Map

- `contracts/`: reusable templates and experiment-log shapes
- `presets/`: current loop presets that are safe to run
- `candidates/`: skills or loop additions being considered for promotion

Use `catalog/skills.yaml` to see whether a skill is:

- part of the default loop
- conditional inside the default loop
- preset-only
- still a candidate

## Loop Stages

1. choose target unit
2. choose benchmark card
3. choose rubric sections
4. draft or revise
5. judge
6. decide rewrite level
7. apply one intervention
8. compare deltas
9. stop or continue

Use the files in this folder to keep runs comparable.
