# Skill Evaluations

This folder is for evidence about whether a skill deserves a place in the loop.

Use it to store:

- benchmark deltas attributable to a skill
- before/after comparisons
- failure notes
- promotion or rejection decisions

## Validation Status Meanings

- `unreviewed`: no real loop evidence yet
- `trialing`: in active use, but not yet justified by enough benchmark evidence
- `validated`: has enough repeatable evidence to keep its slot by default

See [catalog/skills.yaml](/Users/setavya/Hacks/skills/catalog/skills.yaml) for the current status of each skill.
