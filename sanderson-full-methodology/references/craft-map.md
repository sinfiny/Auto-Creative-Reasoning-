# Craft Map

This map condenses the full Brandon Sanderson lecture series into one practical operating model.

## Phase 1: Process And Posture

- treat process as discoverable
- finish books to learn what kind of writer you are
- use outlining and discovery as tools, not identities
- revise deliberately rather than compulsively

## Phase 2: Promises

- define tone promise
- define genre promise
- define plot question
- define character promise
- define world or magic promise

If the user cannot name these, the story is not ready for complex diagnosis.

## Phase 3: Progress

- show visible movement
- use try-fail cycles
- escalate cost, knowledge, and commitment
- make scenes change the story state

If the book drags, progress is probably invisible or repetitive.

## Phase 4: Structure

- assign jobs to acts and sequences
- build nested arcs at scene, chapter, act, and book scale
- use reversals and reveals when context makes them meaningful

## Phase 5: Character

- tune proactivity
- tune capability
- tune relatability or fascination
- choose the right arc type
- define the late proof action

## Phase 6: Speculative Design

- decide how much the reader must understand
- choose hard, soft, or mixed system behavior
- build costs and limitations
- make world detail alter choices and stakes

## Phase 7: Endings

- pay off the right promise
- use seeded tools fairly
- keep climax character-centered
- let the final choice reveal the protagonist

## Phase 8: Revision

- gather feedback
- detect recurring pain points
- diagnose cause before implementing fixes
- revise one layer at a time

## Phase 9: Publishing

- separate craft from business
- choose path by goals and tradeoffs
- research real operators
- build direct audience assets where possible
