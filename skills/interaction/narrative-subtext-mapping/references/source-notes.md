## Fiction Translation

Subtext fails when the agent writes only from surface text. The remedy is to make the invisible character graph explicit before writing:

- What each character knows.
- What each character wants but cannot admit.
- What each character truly feels.
- What each character uses as a visible tactic.
- What the reader can infer from the mismatch.

The hidden map is not the prose. It is the cognitive tool that lets the prose remain indirect.

## Scene ECS Mapping

Useful components for an AI writing IDE:

```text
CharacterKnowledge: confirmed facts, mistaken beliefs, secrets, suspicions
HiddenAgenda: desired outcome of the exchange
TrueEmotion: fear, desire, envy, shame, grief, love, resentment, relief
VisibleTactic: joke, deflection, command, caretaking, silence, accusation, formality
SubtextLeak: body action, word choice, object handling, timing, overreaction
ReaderInference: what the reader should infer without being told
```

Dialogue should be generated from the gap between `TrueEmotion` and `VisibleTactic`.
