# Brandon Sanderson Writing System

This package turns Brandon Sanderson's 2025 writing lecture playlist into a reusable study and execution system.

It is built for two use cases:

- Learn the method lecture by lecture without watching the videos.
- Trigger one narrow craft skill or route a project through the full methodology.

## Included Docs

| File | Purpose |
|---|---|
| `lecture-notes.md` | Lecture-by-lecture important points, skill opportunities, and workflow opportunities |
| `workflows.md` | Reusable workflows derived from the lectures |
| `skill-finalization.md` | Final skill map, why each skill exists, and when to trigger it |

## Included Skills

| Skill | Covers |
|---|---|
| `sanderson-professional-writing` | mindset, process, finishing, practice, revision discipline |
| `sanderson-promise-progress-payoff` | promises, momentum, payoff, plot diagnosis |
| `sanderson-story-architecture` | structure, scene/act jobs, try-fail cycles, nested arcs |
| `sanderson-character-engine` | proactivity, capability, relatability, customization, arc types |
| `sanderson-speculative-design` | magic systems, worldbuilding, costs, consequences, reader orientation |
| `sanderson-publishing-strategy` | traditional vs indie, agents, conferences, platform, newsletters, rights |
| `sanderson-endings-and-revision` | endings, payoff design, revision loops, final polish |
| `sanderson-full-methodology` | orchestration skill that routes a task through the full system |

## Source Basis

The system is grounded in the 16-video 2025 lecture playlist:

1. Professional writing philosophy
2. Promise, progress, payoff
3. Story structures
4. Plot Q&A
5. Plot FAQ
6. Proactive, relatable, capable characters
7. Customizing character
8. Types of character arcs
9. Sanderson's laws of magic
10. Tools to make worlds better
11. How to worldbuild on Earth
12. Worldbuilding Q&A
13. How publishing works
14. What publishing does for you
15. Demystifying the publishing process
16. Final Q&A on ending, revision, and breaking in

The local playlist downloads live in `../sanderson-2025-playlist/videos/`.

## Recommended Use

- Use `$sanderson-full-methodology` when you want a full-stack story workflow.
- Use the component skills when you already know the problem area.
- Start with `lecture-notes.md` if you want the study guide first.
