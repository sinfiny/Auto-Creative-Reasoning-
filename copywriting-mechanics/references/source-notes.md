# Source Notes

This skill is mainly grounded in the Harry Dry copywriting interview summarized in `David-Perell-writing-skills-proof.md`.

The key reusable tests are:

- can the reader visualize the claim?
- can the claim be falsified?
- could any competitor say the same thing?

The workflow is useful because it turns copy quality into checks instead of vague taste. Keep the skill focused on commercial language, not literary style.
